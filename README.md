# CheerpJ Demo: File Upload & Download

This demo is a simple note taking app which is intended to show the file upload and download functionality in CheerpJ.

## Run Locally

```bash
git clone https://github.com/leaningtech/cheerpj-demo-file-upload-download-applet.git
cd cheerpj-demo-file-upload-download-applet
npx http-server
```
